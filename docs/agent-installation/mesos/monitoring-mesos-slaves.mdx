    <li>Keep Middleware API Key and Target at hand</li>

## Install Middleware Agent - on Mesos Slave

### Download our DEB file
```bash
wget -O mw-agent_1.0.0-1_amd64.deb https://github.com/middleware-labs/mw-agent/releases/download/1.0.0/mw-agent_1.0.0-1_amd64.deb
```

### Install DEB file
```bash
sudo dpkg -i mw-agent_1.0.0-1_amd64.deb
```

### Start Middleware Agent
```bash
mw-agent start --api-key jpawpxwzsiwekdyzgolwnonkbmxzuagqfclu --target https://keval.middleware.io:443
```

## Uninstall Middleware Agent
```bash
sudo dpkg -r mw-agent
```
