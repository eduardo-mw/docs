# Prerequisites

<Steps>
    <Step title="Java APM">
        The Java APM must be installed prior to installing Scala. Learn how to install the Java APM [here](/apm-configuration/java)
        <Note> If you have already installed the Java APM, make sure you have the most up to date Middleware Jar file from the <a href="https://github.com/middleware-labs/opentelemetry-java-instrumentation/releases"> Github release page.</a> </Note>
    </Step> 

    <Step title="Scala Version">
        `Scala version 2.11.6`. Check your Scala version with the following command:
        ```bash Shell
        scala -version
        ```
    </Step>

</Steps>

# Install

<Accordion title="Step 1: Update Your `build.sbt` file">
 
 ```java Java
 lazy val root = (project in file("."))
  .settings(
    ...
    (run / javaOptions) ++= Seq(
      "-javaagent:middleware-javaagent-{VERSION}.jar",
      "-Dotel.service.name={SCALA-APP-NAME}"
    )
    ...
  )
 ```

</Accordion>

<Accordion title="Step 2: Run Your App">
Run the following command in your terminal to start your application

```shell Shell
MW_API_KEY=<YOUR_API_KEY> sbt run
```
</Accordion>
