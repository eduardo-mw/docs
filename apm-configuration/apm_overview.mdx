# FAQ

<Accordion title="What are the prerequisites for installing APMs?">
  Prerequisites may vary depending on the APM you're installing. Generally, you'll need access to the application environment, specific permissions, and compatibility with the APM technology.
</Accordion>

<Accordion title="Do I need to configure additional settings after installation?">
  Yes, additional setup may be required, such as initializing container variables, capturing application data, enabling logging, importing trackers, or implementing an APM collector. Refer to the respective APM installation guides for detailed instructions.
</Accordion>

<Accordion title="How do APMs help me improve application performance?">
  APMs provide real-time data on your application's performance, helping you identify and resolve issues efficiently. This, in turn, leads to better user experiences and optimized resource usage.
</Accordion>
