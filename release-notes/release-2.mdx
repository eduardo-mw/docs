# Features
- PHP Azure Build Pipelines
- [RUM Mobile - React Native](https://docs.middleware.io/rum/rum-react)
- Spend Analysis charts
- Trace correlation drill down from specific log messages
- Price calculator that analyzes your bill based on the usage of different products
- Sort and group attributes of log-collected data

# Changes
- Billing tab now displays information about plan name, billing date, billing address, payment details and option to update billing information
- Invoice tab now displays all past invoices with invoice date, paid date, and the option to download invoices
- New default dashboard for Kubernetes with pod and cluster overview metrics

# Fixes
- Fixed column sorting issue in log listing
- Sort logs based on timestamp
- Enable/disable metric and trace collection at the host level
