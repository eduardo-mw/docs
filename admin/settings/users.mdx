# Overview

The Users section is where you manage members in your organization. You can add new teammates, grant admin privileges, and filter how you view your active users.

<Note> For more information on the difference between Users and Admins, please refer to the [Roles and Permissions](/admin/settings/role-&-permission) page. </Note>
