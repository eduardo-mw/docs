# Overview

The Usage section provides an overview and summary of users historical usage history. You can access data from your Infra, APM and Database Hosts, Synthetic Checks, Serverless Traces, Logs, and RUM while filtering said Usage by date and project type. Data can be downloaded locally via `.csv`.

# Access Usage

## Step 1: Navigate to Settings

## Step 2: Select Usage

## Step 3: Filter by Date
